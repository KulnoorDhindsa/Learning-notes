# Networking

Notes from studying networking fundamentals, building toward practical application in CTFs and homelab work.

## Contents
- `osi-model.md` — the 7-layer conceptual model, how layers map to real traffic
- `tcp-ip.md` — the practical 4-layer model actually used on the internet, and how it differs from OSI
- `http.md` — the application-layer protocol behind web traffic: methods, status codes, headers
- `dns.md` — how domain names resolve to IPs, why it's a separate system from HTTP
- `subnetting.md` — IP addressing, CIDR notation, subnet math
- `ports-and-protocols.md` — quick-reference table of common ports and what runs on them
